Connect Four

Classic 7-column, 6-row connect-four game. First to connect four discs of the same color wins.

Controls:
- Click a column to drop your disc.
- Reset to start a new game.

Persistence:
- Wins are stored in localStorage under `connect-four-wins`.

Notes:
- Initial implementation is single-device 2-player only.
- Future enhancements: player vs CPU, animations, undo, better win highlight.
